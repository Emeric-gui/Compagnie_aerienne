# WondAirlines

## 1. Intitulé
  Application web d'une compagnie aérienne canadienne factice.<br>
  Possibilité de réserver son trajet pour des voyages entre les différents aéroports du Canada.
  
  
## 2. Prérequis
PHP 7<br>
PostgreSQL sur Apache
